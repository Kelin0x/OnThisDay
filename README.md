# 📅 历史上的今天


每天自动更新的历史上的今天事件集锦，让我们一起回顾历史的长河

</div>

## 🌟 特色功能

- 🔄 每日自动更新历史事件
- 📚 涵盖中外历史大事记
- 🎯 精选有趣历史故事
- 🤖 基于 GitHub Actions 自动化
- 🌐 可靠数据源实时更新

## 🛠️ 技术实现

- **数据获取**: Python 爬虫自动采集
- **自动化部署**: GitHub Actions
- **数据存储**: GitHub Repository
- **展示方式**: Markdown 动态渲染

## 📊 更新时间

- 自动更新频率：每天 `00:00` (北京时间)
- 手动更新：支持通过 Actions 手动触发
